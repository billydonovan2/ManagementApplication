# ManagementApplication
This java program was developed by Bill Donovan, Nelson Costa, and Andrew Kim. This program enables the user to set events and appointments on specific dates. The events and appointments can either be a one-time, daily, monthly, or yearly occurrence. By choosing a date, the program will list out all of the appointments of that day accordingly.
